Lines Element

Description

The hr tag defines a thematic break in an HTML page (e.g. a shift of topic).

Syntax

Start Tag: <hr> End Tag: </hr>

Example

<h4>HTML</h4> <p>HTML is a language for describing web pages.</p> <hr> <h4>CSS</h4> <p>CSS defines how to display HTML elements.</p>

Display

HTML

HTML is a language for describing web pages.

CSS

CSS defines how to display HTML elements.

Comments Element

Description

The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers.

Syntax

Start Tag: <!-- --> End Tag: </!-- -->

Example

<!-- This is a comment --> <p>This is a paragraph.</p> <!-- Comments are not displayed in the browser -->

Display

This is a paragraph.
